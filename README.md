* https://wonseok-han.dev

## 개발 환경

* **NodeJS:** `v22.11.0` (LTS)
* **PackageManager:** `npm`

## Getting Started

### 실행

```bash
# Module Install
npm install

# Server start
npm run dev
```

### 배포

* Build 명령 후 생성되는 .next 폴더로 배포합니다.

```bash
# Build
npm run build
```
